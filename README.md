# 개발 경력기술서 프로필

개발자 프로필을 보여주는 정적 웹사이트입니다.

## 📋 프로젝트 소개

이 프로젝트는 개발자의 경력, 기술 스택, 프로젝트 경험 등을 보여주는 프로필 페이지입니다.

## 🚀 배포된 사이트

GitHub Pages를 통해 배포된 사이트: [프로필 보기](https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/etteuProfile.html)

## 📁 파일 구조

```
.
├── etteuProfile.html    # 메인 프로필 페이지
├── profile-data.json    # 프로필 데이터 (JSON 형식)
├── script.js            # 프로필 렌더링 로직
├── styles.css           # 스타일시트
└── img/                 # 이미지 파일들
```

## 🛠️ 사용 방법

1. `profile-data.json` 파일을 수정하여 프로필 정보를 업데이트합니다.
2. 변경사항을 Git에 커밋하고 푸시합니다.
3. GitHub Pages가 자동으로 업데이트됩니다.

## 📝 데이터 형식

프로필 데이터는 `profile-data.json` 파일에 JSON 형식으로 저장됩니다.

주요 섹션:
- **profile**: 기본 정보 (이름, 이메일, 전화번호, 프로필 이미지 등)
- **introduce**: 자기소개
- **skill**: 기술 스택
- **experience**: 경력
- **project**: 프로젝트
- **opensource**: 오픈소스 기여
- **education**: 학력
- **etc**: 기타
- **article**: 아티클
- **coverLetter**: 자기소개서
- **portfolio**: 포트폴리오
